## Permission levels

There are 4 types of permission levels in the product.

| Role          | Capabilities                            |
| ------------- | --------------------------------------- |
| Administrator | Has all admin privileges                |
| Editor        | Can edit posts                          |
| Viewer        | Can only view posts                     |
| Guest         | Can only view posts they are inivted to |
